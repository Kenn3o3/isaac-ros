# System Requirement:

OS: Ubuntu 22.04

# Prerequistes:

Install MkDocs and Necessary Plugins(pip)
```
pip install mkdocs
pip install mkdocs-material
```

Change site_url in mkdocs.yml
```
site_url: https://<Your github username>.github.io/isaac-ros/
```

# Build and Serve Your Website Locally
```
cd /isaac-ros
mkdocs serve
```
